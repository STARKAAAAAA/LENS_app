1. Understanding Refraction

Snell–Descartes law: n₁ sin(θ₁) = n₂ sin(θ₂). Drag the sliders to see how light bends at different refractive indices.

2. Creating the Glass Surface

Convex Circle: y=√(1−(1−x)²) — spherical dome
Convex Squircle: y=⁴√(1−(1−x)⁴) — Apple's preferred shape
Concave: y=1−Convex(x)
Lip: y=mix(Convex,Concave,Smootherstep(x))

const derivative = (f(x+δ) - f(x-δ)) / (2δ);
const normal = { x: -derivative, y: 1 }; // rotated −90°
const displacement = height × tan(asin(sin(θ₁)/1.5));

The displacement magnitude is pre-calculated on one radius (127 samples), then rotated around the shape for radial symmetry.
